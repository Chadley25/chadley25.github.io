Hey, if you couldn't already tell, my name's Bradley Chavis, and I'm currently studying in Network Security and Computer Science at UAT. One of these competitions being CCDC (Collegiate Cyber Defense Competition), where I was one of the people in charge of managing and defending Unix machines and eCommerce applications. But when all of that stuff isn't going on, I'm working on personal projects. One of these being H@xc0rd which I'm currently working on along with my partner, David Austad. This project is a Discord bot that will allow anyone with a Discord account to use tools commonly used in the network security field. Also, I have other projects like a vulnerable environment that can be fully built with an Ansible playbook, and a mobile application which will search for where you can stream or purchase a show or movie, called Found It! But that's it for everything you need to know me. Feel free to look around at everything else on my websites, and contact me if you want more information on anything.
