Vulnerable Environment

A virtual environment made to look like one used by a small company, but extremely vulnerable to things like EternalBlue and Drupalgeddon2. This environment contains over 10 virtual machines and more within an ESXi host, all of which are fully functioning and set up for semi-realism. Furthermore, the environment also includes a scoring engine which scores most of the important services located on the machines.

Simple Log Parser

A simple command-line program which will parse through any log file and display a summary of all the information in the end. This information will vary depending on the type of log file specified, though will include the number of lines in the log file, the time frame of all the logs, the average amount of logs in time frames (can be specified in the settings), and more!

All-In-One

A web-based program that encompasses everything that you may use on a daily basis into one giant thing.

Winter-Themed Hangman

A GUI-based hangman game built in C# that anyone can play. The words for this game, and everything else, will be fully winter-themed and pull from a source which can easily be added to.
