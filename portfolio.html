H@xc0rd

A bot for Discord that will provide users with access to tools commonly used in the Network Security field without the need to understand Linux. For each tool, it will come with it's own help menu which will contain all arguments that you can use, a description of the tool, and what the data you get from the tool can be used for.

Vulnerable Environment

A virtual environment made to look like one used by a small company, but extremely vulnerable to things like EternalBlue and Drupalgeddon2. This environment contains over 10 virtual machines and more within an ESXi host, all of which are fully functioning and can be set up through the provided Ansible playbook.

Simple Log Parser

A simple command-line program which will parse through any log file and display a summary of all the information in the end. This information will vary depending on the type of log file specified, though will allways include the number of lines in the log file, the time frame between the first and last log, and more!
