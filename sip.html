H@xc0rd

H@xc0rd

The Educational Toolset

H@xc0rd, which is soon to be open source, is a Discord bot built upon the Discord.js library that provides users with tools commonly used in the Network Security field without the need to understand Linux. These tools range from exploit to recon tools, consisting of some such as hashcat, hydra, nmap, Shodan, and many more. And along with all of these tools, each one will come along with a description of the tool, all arguments that can be used within the tool, and an explanation of how the data gathered from the tool could be used. While this project was origionally intended for begginers in the Network Security field, it has evolved and is now being built to be used by anyone, no matter their skill level.

Network Security Degree Objectives

Objective 1

Create a network infrastructure design communications document that includes identified hardware components, connections to outside world, identified physical layer connectivity (media) and addressing, including operational and security components in the design.

Artifact 1 - Home Network Design

A network infrastructure design that's mean to mimic what can be done with common at home network hardware.

Artifact 2 - Segmented & Redundant Network Design

A network infrastructure design that highlights some of the most common security practices within a segmented network, including redundancy.

Objective 2

Install, configure and test security hardware and software tools with supporting documentation such as port scanners, vulnerability detection systems, intrusion detection systems, firewalls, system hardening, anti-virus tools, patch management, auditing and assessment.

Artifact 1 - pfSense & Surricata

An implementation of both pfSense and Surricata set up in a hypervisor on a Windows Server.

Artifact 2 - Steel Mountain

A TryHackMe writeup where I utilized multiple tools including Nmap, Nikto, and Gobuster to find information about the machine.

Objective 3

Construct, implement and document a script or a program to automate a security related process or other tasks such as installation, administration, management, mapping resources, logon scripts, patch management, updates, auditing, analysis and assessment.

Artifact 1 - Basic Linux Hardening

A Bash script that hardens multiple services commonly found on Linux machines.

Artifact 2 - Shell Finder

A Python program that finds and kicks out shells that are gained in the midst of an attack.

Objective 4

A presentation about the implementation of SP 800-76-2 into a business.

Artifact 2 - Security Policies

A presentation about the implementation of security policies within a business.

Objective 5

Develop a research report or implementation plan concerning legal and ethical best practices and mandated requirements that pertain to information security.

Artifact 1 - AI

A writeup about the integration of AI into security and all the issues related to it.

Artifact 1 - SP 800-76-2 Implementation Plan

A presentation about the implementation of SP 800-76-2 into a business.

Objective 6

Research, document, test and evaluate several current industry information security based threats, risks, malicious activities, covert methodology, encryption technologies, mitigation techniques or unconventional tactics to prevent loss of sensitive information and data confidentiality, integrity and availability.

Artifact 1 - GlueBall

A presentation about what GlueBall is and how it affected Windows systems for over 2 years.

A writeup of the ZeroLogon vulnerability explaining what it is and how it affected nearly everyone.
