Hey, welcome to my portfolio website! If you didn't know already, my name's Bradley Chavis and I'm currently studying in two majors of Network Security and Computer Science at UAT. From my five semesters here so far, I've learned a lot of things through the classes I've taken, the projects I've done, and the competitions I've competed in. One of these competitions being CCDC (National Collegiate Cyber Defence) where I was one of the people in charge of defending Unix machines and managing eCommerce applications. And when that's not going on, I'm working on my SIP (student innovation project) along with my partner, David Austad. This project is pretty much a Discord bot that will allow anyone with a Discord account to use tools commonly used in the network security field on any device through Discord. Each of these tools not only will come with their own custom-built code, allowing for any arguments to be used that could normally be used within a shell but will also come with a help menu that will explain the tool in-depth and display arguments that could be used.

Note: This website is still being worked on, so don't expect the code to look beautiful or all information to be added yet.
