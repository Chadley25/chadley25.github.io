H@xc0rd

The Educational Toolset

H@c0rd, which is soon to be open source, is a Discord bot built upon the Discord.js library that provides users with tools commonly used in the Network Security field. These tools range from anything like hashcat, hydra, nmap, Shodan, and many more. And along with all of these tools, each one will come along with a description of the tool, all arguments that can be used within the tool, and an explanation of how the data gathered from the tool could be used. While this project was origionally intended for begginers in the Network Security field, it has evolved and is now being built to be used by anyone, no matter their skill level.
